#查询
##查询基本语句
```sql
select[ALL|DISTINCT]<目标列表达式>[,<目标列表达式>]···
from<表名或视图名>[,<表名或视图名>]
[where<条件表达式>]···
[group by <列名字>[HAVAING<条件表达式>]]
[order by[ASC|DESC]]
//group by进行分组处理
//order by进行排序处理
```
##2单表查询
###1.选择表中若干列

```sql
//查询所有学生姓名及所在学院
select sname,college
from s
//单击执行
```
####（2）查询全部列
```sql
//查询所有课程信息
select ALL
from c
```
